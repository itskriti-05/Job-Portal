# 💼 Job Portal (MERN)

A fully responsive **Job Portal Application** built using the **MERN stack (MongoDB, Express, React, Node.js)**.  
It connects **Job Seekers** and **Employers** with role-based access, job posting, applications, and resume upload features.

---

## 🚀 Features
- 🔐 Authentication & role-based access  
- 👤 Job seekers: browse, apply, save jobs, upload resume  
- 🏢 Employers: post jobs, manage listings, view applicants  
- 📊 Dashboards for both roles  
- 📱 Responsive UI  

---

## 🛠️ Tech Stack
- Frontend: React (Vite), CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB
